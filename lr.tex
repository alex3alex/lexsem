\frametitle{Semantic Resources}

\begin{block}{Definition}

A \textbf{semantic resource} is an undirected graph $(C, R)$:
\begin{itemize}
\item nodes $C$ represent \textbf{terms};
\item edges $R$ represent untyped \textbf{semantic relations}.
\end{itemize}
